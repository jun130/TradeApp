# TradeApp

Trading application with people

<h3>Function</h3>

1. Sing Up, Login, Logout
2. Write and upload item image to board
3. Display my information, category, item list
4. Item add, remove, search
5. Chatting


<h3>Development Environment</h3>

- OS: Windows 10
- IDE: Android Studio
- DB: Firebase
- Language: Java
